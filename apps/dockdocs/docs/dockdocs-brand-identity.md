# DockDocs Brand Identity System

## Recommended Direction

Direction A: AI Workspace.

The mark combines a document outline, folded page corner, structured text lines, and a restrained blue intelligence node. It positions DockDocs as an AI document workspace rather than a commodity PDF tool.

## Concepts Reviewed

### Direction A: AI Workspace

- Logo Concept: compact document workspace mark with a small AI node.
- Icon Concept: dark rounded square, white document stroke, blue intelligence point.
- Brand Symbol: document plus node, recognizable without the wordmark.
- Color Direction: white, black, neutral gray, restrained blue.
- Typography Direction: Inter or Geist-style geometric SaaS sans.

### Direction B: Document Intelligence

- Logo Concept: folded document with extraction lines and analysis layers.
- Icon Concept: structured scan frame inside a document.
- Brand Symbol: document plus intelligence grid.
- Color Direction: neutral foundation with blue analysis accent.
- Typography Direction: precise, editorial SaaS sans.

### Direction C: Knowledge Operating System

- Logo Concept: modular workspace window with document cells.
- Icon Concept: operating-system tile grid.
- Brand Symbol: knowledge grid with one active node.
- Color Direction: monochrome UI system with subtle blue status.
- Typography Direction: utilitarian product sans.

## Usage Guidelines

- Use the symbol at 16x16 and 32x32 in browser contexts.
- Use the symbol plus wordmark in the header.
- Keep the blue node as the only accent in the mark.
- Use the mono mark when color reproduction is unavailable.
- Do not place the mark on noisy gradients or photo backgrounds.
- Do not replace the symbol with a generic PDF page icon.
