# SEO-010 · 多语言母语关键词目标(es 优先 + zh 参考)

> **为什么**:多语言 SEO 最大的杠杆是——每种语言要用**母语真实搜索词**做标题/H1/描述/FAQ,而不是英文词的直译。GSC 已出现 `ppt a pdf`(西语,49名)= 西语流量在进、母语词路子正确。
>
> **车道**:本清单是 SEO-GEO 窗口的研究产出;**应用到 es 标题/FAQ 的是"多语言"窗口**(`localized-tools.ts` 的 `esTools` / `esFaq`)。已知短板:14 个 es 工具页目前是"西语界面 + 英文 SEO 元数据"。
>
> **怎么用**:把每个工具页的 es `title` / `heroTitle` / `keywords` / FAQ 对齐到下表"首选词";一个标题里自然带上 1-2 个变体即可,不要堆砌。
