# Manual GSC Steps

1. Open Google Search Console property for `https://dockdocs.app/`.
2. Confirm `https://dockdocs.app/sitemap.xml` shows Submitted / Success.
3. Day 1: inspect only P0 URLs and request indexing where GSC allows it.
4. Day 2-3: inspect the first 20 P1 URLs. Do not submit all P1 URLs at once if GSC starts rejecting requests.
5. Day 4-7: monitor Coverage / Pages / Sitemaps / URL Inspection results.
6. Do not manually submit P2 until at least some P0/P1 URLs show crawl or index movement.

# What To Do If Request Indexing Is Rejected

- Do not repeatedly retry the same URL in the same session.
- Confirm the URL returns HTTP 200.
- Confirm the URL is in sitemap and matches canonical exactly.
- Confirm no `noindex` appears in URL Inspection.
- Try a different P0 or P1 URL instead of brute-forcing requests.
- Wait 24 hours before retrying rejected high-priority URLs.
- If many requests are rejected, stop manual requests and let sitemap discovery work for 3-7 days.
